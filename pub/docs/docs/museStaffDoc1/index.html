MuseStaff

Constructor

Creates a new MuseStaff object.

Syntax

MuseStaff(timeSignature)

Parameters

ParameterDescriptionRequired
timeSignatureString format of time signature (e.g. "3/4"). If not provided, the default time signature "4/4" is used.Optional

Options

All the global options available.

PropertyDefaultTypeDescription
timeSigTimeSignature(4,4)TimeSignatureThe time signature for the staff. Restricts the number and size of notes allowed in each measure of the staff.
editablefalsebooleanIf the staff is in "edit-mode", where you can dynamically click the staff to add notes.
measures[ ]Array[Measure]Array of all the measures in the staff. The order of the Measure objects correspond to the order of the measures displayed in the staff.
measurePointedOnnullMeasureIf the staff is editable, the staff has a reference to which measure and note the pointer is positioned at.
staffnull<div class="museStaff">The staff display element.
insertNoteType"quarter"StringThe unit type of the current note that is selected to be inserted while in "edit-mode". MuseStaff.insertNoteType can be one of the following: "whole", "half", "quarter", "eighth", "sixteenth".

Displaying MuseStaff on the front-end

MuseStaff.staff would be the element that you would add to your DOM. For example, after creating a MuseStaff, for it to display on the screen, you can append MuseStaff.staff to the body of an html page:

const museStaff = new MuseStaff();
museStaff.addMeasure();

// Body element in html
const body = document.querySelector("body")
body.appendChild(museStaff.staff)
