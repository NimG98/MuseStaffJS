MuseStaff

Methods

addMeasure

Creates a new Measure and adds it to the staff. If measureIndex is provided, insert the measure into the staff at that index. Otherwise, insert the measure at the end of the staff.

Syntax

addMeasure(measureIndex)

Parameters

ParameterDescriptionRequired
measureIndexIndex to insert Measure object into MuseStaff.measuresOptional

setNotes

Set the staff's notes to be the ones provided in the array of Note objects, notes. If the staff is not empty (i.e. filled with notes beforehand), then the staff is emptied before setting the staff's notes to the ones in notes.

Syntax

setNotes(notes)

Parameters

ParameterDescriptionRequired
notesArray of Note objects to be placed in the staff.Required

setEditable

Set the staff to be editable or not. If the staff is editable, (i.e. in "edit-mode"), the staff can be clicked to insert and modify notes. Otherwise, the staff just displays the current notes in the staff's Measure objects.

Syntax

setEditable(editable)

Parameters

ParameterDescriptionRequired
editableBoolean indicating if the staff should be editable.Required
